The solution
Input Generation Evaluation Feedback loop

An expert system guided LLM generates neural architectures tailored to user needs

User requirements and domain rules are first interpreted by an expert system and converted into structured prompts for a large language model. The model generates candidate architectures, which are evaluated across multiple objectives, and the feedback is used to iteratively refine designs to match user-specific constraints.
